King Shadow - Server Rules

القوانين العامة

  • احترام جميع اللاعبين والموظفين واجب.
  • ممنوع السب أو الإهانة أو السخرية.
  • الالتزام بالـ RP وعدم التخريب في اللعبة.
  • الالتزام بالقوانين لضمان بيئة ممتعة للجميع.

قوانين الرول بلاي (RP)

  • ممنوع الخروج عن الشخصية بأي شكل.
  • ممنوع الميتاقيمينق أو الباور قيمينق.
  • يجب تقمص الشخصية بشكل واقعي ومناسب.
  • التفاعل مع الأحداث بشكل منطقي داخل اللعبة.

قوانين الإجرام

  • ممنوع القتل أو السرقة العشوائية.
  • يجب وجود سبب منطقي لأي جريمة.
  • الالتزام بعدد محدد للسرقات أو الجرائم حسب القوانين.
  • عدم استغلال الثغرات أو الأخطاء التقنية.

قوانين الإدارة

  • احترام قرارات الإدارة أمر إلزامي.
  • الاعتراض أو النقاش يكون عبر القنوات الرسمية فقط.
  • أي شكوى يتم رفعها عبر دسكورد السيرفر.
